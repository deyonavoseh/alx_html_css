Introduction

This webpage is a simple HTML example created as part of a learning project. It demonstrates the use of headings, paragraphs, and images.

HTML stands for HyperText Markup Language and is the standard markup language for creating webpages. It allows us to structure content such as text, images, and links.

Adding headings and paragraphs helps organize content, making it easier to read and navigate. Headings come in levels from <h1> (most important) to <h6> (least important).
